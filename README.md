# Sales-Analytics-Dashboard
Built an interactive Sales Analytics Dashboard using Microsoft Power BI to monitor key business metrics including sales, customers, and transactions. Enabled data-driven decision-making by analyzing product performance, customer segments, and sales trends through dynamic visualizations.
## Dashboard Overview
----------
      The dashboard helps analyze customer behavior, product performance, and sales trends using multiple visualizations like KPI cards, charts, and slicers. It enables users to quickly understand business performance and make data-driven decisions.
  
   ##   KPI Cards (Top Metrics)

•	Total Sales 

•	Customer Count (Customer ID) 

•	Average Sales 

•	Total Transactions 

•	Age Bucket Size 

These KPIs provide a quick summary of business performance.

## 📊 Visualizations Used

•	📈 Bar Chart → Product Category vs Sales 

•	🥧 Pie Chart → Gender vs Sales 

•	🍩 Donut Chart → Customer Type vs Customer Count 

•	📊 Bar/Column Chart → Age Bucket vs Sales 

•	📋 Table → Age, Product, Sales 

## 🎛️ Filters / Slicers

•	🛍️ Product  Category

•	👥 Customer Type 

These slicers allow dynamic filtering and better exploration of data.

## ❓ Business Questions

1.	Which product category generates the highest sales?

3.	Which gender contributes more to overall sales? 
4.	How do sales vary across different age groups? 
5.	Which customer type brings more transactions?

            The Sales Analytics Dashboard provides a comprehensive view of business performance by combining key metrics,
